# KirSDK

KirSDK — это программный пакет, предназначенный для интеграции с инструментами разработки от Kirillovo-Company.  
Он автоматизирует установку и настройку необходимых компонентов, начиная с клонирования репозитория `PyCode`.

---

## Требования

Для корректной работы установщика `installKirSDK.py` требуется:

- Python 3.7 или выше
- Установленный пакет `GitPython`

Установить `GitPython` можно с помощью pip:

```bash
pip install GitPython
```

---

## Установка

1. Скачайте файл `installKirSDK.py`.
2. Убедитесь, что у вас установлен `GitPython` (см. раздел выше).
3. Запустите скрипт:

```bash
python installKirSDK.py
```

Скрипт выполнит следующее:
- Создаст директорию `PyCode`, если она отсутствует.
- Клонирует репозиторий [https://github.com/Kirillovo-Company/pycode](https://github.com/Kirillovo-Company/pycode) в папку `PyCode`.
- Сообщит о завершении установки.

---

## Примечания

- Убедитесь, что у вас есть доступ к интернету и разрешение на клонирование репозитория по HTTPS.
- Если папка `PyCode` уже существует, содержимое не будет перезаписано — скрипт не удаляет существующие данные.
